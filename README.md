# 🎀 Outubro Rosa: Prevenção e Conscientização

Este projeto foi desenvolvido como um exercício prático de **HTML5 Semântico** e **Sass (SCSS)** para a turma **Empower 5.0 da Vai na Web**. O objetivo é criar uma página de destino (landing page) focada na conscientização sobre a prevenção e o diagnóstico precoce do câncer de mama.

Através deste tema de grande relevância social, exploramos técnicas de layout flexível e acessibilidade no desenvolvimento web.

---

## 🚀 Tecnologias & Ferramentas

* **HTML5**: Estruturação semântica do conteúdo (uso de `header`, `main`, `section`, `nav` e `footer`).
* **Sass (SCSS)**: Pré-processador CSS utilizado para organizar estilos com aninhamento e variáveis.
* **Google Fonts**: Integração da família tipográfica "Noto Sans" para melhor legibilidade.
* **Responsive Design**: Implementação de conceitos de `viewport height (vh)` e `viewport width (vw)`.

---

## 🎯 Objetivos Pedagógicos

Nesta aula, focamos em como construir uma interface limpa e informativa, trabalhando os seguintes pontos:

* **Semântica Estrutural:** Organização correta das tags para SEO e leitores de tela.
* **Flexbox Layout:** Uso intenso de `display: flex`, `justify-content` e `align-items` para alinhar textos e imagens de forma harmônica.
* **Mobile First & Breakpoints:** Adaptação completa do layout para telas menores (tablets e smartphones) utilizando `@media` queries.
* **Estilização com SCSS:** Prática de aninhamento de seletores para um código CSS mais limpo e fácil de manter.

---

## 📂 Estrutura do Projeto

* `index.html`: Estrutura principal da página com todas as sessões de alerta, corrida rosa e dicas de saúde.
* `style.css`: Arquivo compilado a partir do SCSS com as regras de design e responsividade.
* `/Imgs`: Diretório contendo todas as peças visuais que compõem a identidade do projeto.

---

## 📱 Responsividade na Prática

O projeto foi planejado para se transformar em diferentes telas:
* **Desktop:** Layout horizontal, aproveitando o espaço para colocar textos ao lado de imagens.
* **Mobile (max-width: 768px):** Transição para o modo vertical (`flex-direction: column`), garantindo que a leitura seja confortável e os elementos não fiquem espremidos.
